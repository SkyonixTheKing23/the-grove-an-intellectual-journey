# 🌱 What is this?

**The Grove** is a story-driven personality quiz inspired by **pixel RPG aesthetics**.

Instead of a plain form, you're walked through questions by two animated pixel art characters who each represent a different side of human knowledge.

At the end, you get a **personalised result** with:

- A description  
- A trait line  
- A shareable card you can send to friends

---

# ✨ Features

- **22 questions across three rounds** with character dialogue between each set  
- **Animated 2-frame pixel art sprites** for both characters  
- **Dynamic theme switching** — the entire site colour palette changes based on which character is active  
- **Web Audio API sound effects** — pixel blips, chimes, whooshes, and a fanfare on the result screen, all generated in-code with no audio files  
- **Shareable result card** — animated card with floating pixels, shimmer effect, and colour theming based on your result  
- **Fully responsive** across desktop, tablet, and mobile  
- **No frameworks, no dependencies, no external files** — pure HTML, CSS, and JavaScript
